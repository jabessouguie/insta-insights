# Roles InstaInsights

Ce dossier contient la definition des roles necessaires pour faire evoluer InstaInsights
de son etat de MVP vers un produit SaaS mature. Ces fiches sont a destination des futurs
collaborateurs, co-fondateurs ou freelances qui rejoignent le projet.

> Ce dossier est exclu du depot git (`.gitignore`). Il est uniquement partage en interne.

## Lecture recommandee

Avant de lire une fiche de role, il est recommande de lire :
- `README.md` a la racine du projet (architecture et stack technique)
- `CONTRIBUTING.md` (standards de qualite et conventions de code)
- `ROADMAP.md` (vision produit, priorites et KPIs)
